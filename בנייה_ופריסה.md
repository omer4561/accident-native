## Secrets ב-GitHub

### Requirement
**חובה** — הוגדר לפני build:

| Secret | תיאור | חובה? |
|--------|-------|-------|
| `HF_TOKEN` | Hugging Face token | ✅ כן |
| `TELEGRAM_TOKEN` | Telegram bot token | ✅ כן |
| `CHANNEL_ID` | Telegram channel ID | ✅ כן |
| `GEMINI_KEY` | Google Gemini API key | ⚠️ אופציונלי |
| `EMAILJS_PUBLIC_KEY` | EmailJS public key | ⚠️ אופציונלי |

### Injection בקוד
ב-build-apk.yml, שלב "Set APP_BUILD...":
```bash

sed -i "s|__ADMIN_PASS_HASH__|$ADMIN_HASH|g" www/index.html
```

## Android Release Keystore

### קובץ
`accident-release.keystore` — **זה ממשי קובץ בינארי שהוא **committed** לרפו

### סיסמה
- מאוחסן ב-GitHub Secret: `KEYSTORE_PASSWORD`
- **חובה להגדיר** — fallback hardcoded הוסר

### Generation (ראשון פעם בלבד)
```bash
keytool -genkeypair -v \
  -keystore accident-release.keystore \
  -alias accident \
  -keyalg RSA -keysize 2048 -validity 10000 \
  -storepass "$KEYSTORE_PASSWORD" \
  -keypass "$KEYSTORE_PASSWORD" \
  -dname "CN=Accident Reporter,C=IL" \
  -noprompt
```

### Signing ב-Gradle
build.gradle מתוקן בעת build:
```gradle
signingConfigs {
  release {
    storeFile     file(System.getenv("HOME") + "/release.keystore")
    storePassword "$KEYSTORE_PASSWORD"
    keyAlias      "accident"
    keyPassword   "$KEYSTORE_PASSWORD"
  }
}

buildTypes {
  release {
    signingConfig signingConfigs.release
  }
}
```

## Build Steps בעלייה

### 1. Secrets Injection
כל secrets מ-GitHub מוזרקים לקוד www/

### 2. Tesseract Downloads
```bash
# Tesseract.js libraries
curl -L https://cdn.jsdelivr.net/npm/tesseract.js@5/... → www/tesseract/

# Hebrew trained data
curl -L ...tessdata/heb.traineddata.gz → www/tesseract/tessdata/

# English trained data
curl -L ...tessdata/eng.traineddata.gz → www/tesseract/tessdata/
```

**תוצאה**: OCR עובד offline בלא הורדה בעת runtime

### 3. Capacitor Sync
```bash
npx cap add android     # adds android/ folder if not present
npx cap sync android    # copies www/ to android/app/src/main/assets/
```

### 4. AndroidBridge Injection
Python script מוזרק לתוך MainActivity.java:
- `saveFile()` — MediaStore.Downloads
- `readFile()` — MediaStore query
- `downloadAndInstall()` — APK download + system installer
- `scanFile()` — MediaScanner
- `getInstalledBuild()` — build number קבוע

### 5. Build & Sign
```bash
./gradlew assembleRelease --no-daemon
# Output: android/app/build/outputs/apk/release/app-release.apk
```

### 6. Verify Signature
```bash
apksigner verify --verbose app-release.apk
# Signature: release key
```

## OTA Update System

### היום User Code ב-index.html

#### checkUpdate() — בחזקת החזקת
```javascript
async function checkUpdate() {
  const resp = await fetch(
    'https://omer4561.github.io/accident-native/version.json'
  );
  const data = await resp.json();
  
  if (data.apk > APP_BUILD) {
    // יש עדכון חדש
    showUpdatePrompt(data.apk_url);
  }
}
```

**מזומן:**
- בעלייה אפליקציה (בעלייה עמוד)
- כל שעה (interval)

#### downloadAndInstall() — AndroidBridge
```javascript
// בקריאה מ-index.html
AndroidBridge.downloadAndInstall(apkUrl);

// ב-MainActivity.java:
@JavascriptInterface
public void downloadAndInstall(String apkUrl) {
  // 1. download APK לתוך cache
  // 2. progress tracking → window._apkProg(percent)
  // 3. FileProvider.getUriForFile() (Android 7+)
  // 4. Intent ACTION_VIEW + APK mime type
  // 5. System installer prompt
}
```

### Download Progress
```javascript
window._apkProg = function(percent) {
  document.getElementById('progress').innerText = percent + '%';
};

window._apkErr = function(err) {
  console.error('APK Error:', err);
};
```

AndroidManifest.xml מעודכן בעת build:


## Troubleshooting

### "KEYSTORE_PASSWORD not set"
```
❌ Error: KS_PASS is empty
```
**פתרון:** הוסף `KEYSTORE_PASSWORD` ב-GitHub Secrets

### "APK verification failed"
```
❌ ERROR: VerifyException: Signature not valid
```
**בדוק:**
1. Keystore קיים בפועל
2. סיסמה תכניסה נכונה ב-Secret
3. Java 17 מותקן

### "build-apk.yml failed to patch build.gradle"
```
❌ Python: KeyError: 'signingConfigs not found'
```
**בדוק:**
- android/app/build.gradle יש android { } block
- Capacitor sync רץ בהצלחה
