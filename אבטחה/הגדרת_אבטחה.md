## שלב 1: הוספת GitHub Secrets

### ADMIN_PASS (חובה)
סיסמה לפאנל הניהול.

**הוסף:**
1. GitHub → Settings → Secrets and variables → Actions
2. "New repository secret"
3. Name: `ADMIN_PASS`
4. Value: `<בחר סיסמה חזקה, לפחות 12 תווים>`

**דוגמה:**
```
MySecure@Pass2026!
```

### KEYSTORE_PASSWORD (חובה)
סיסמה ל-Android release keystore.

**הוסף:**
1. GitHub → Settings → Secrets and variables → Actions
2. "New repository secret"
3. Name: `KEYSTORE_PASSWORD`
4. Value: `<בחר סיסמה חזקה, לפחות 12 תווים>`

**דוגמה:**
```
KeyStore@Pass2026!
```

### EMAILJS_PUBLIC_KEY (אופציונלי)
אם רוצה שהרשמה בדוא"ל תעבוד.

1. עבור ל-https://www.emailjs.com/
2. Dashboard → Integrations → Add Service
3. בחר SMTP או שירות מייל אחר
4. Copy את ה-Public Key
5. GitHub Secrets → New
   - Name: `EMAILJS_PUBLIC_KEY`
   - Value: `<ה-key מ-EmailJS>`

## שלב 5: בדוק הגנות OTP

**בדוק בקונסול (DevTools):**

1. הרשם עם מייל/טלפון
2. F12 → Console
3. הקלידו:
   ```js
   JSON.parse(localStorage.getItem('pending_otp'))
   ```
4. **בדוק:** 
   - יש `codeHash` (hex ארוך)?
   - **אין** `code` (קוד גולמי)?
   - ✅ יש `tries: 0`?

## שלב 6: בדוק XSS Protection

**בדוק ב-admin panel:**

1. התחבר ל-admin
2. F12 → Network
3. ראה את ה-HF API calls
4. בדוק ש-HF_TOKEN לא מופיע כ-`?token=...`
5. ✅ צריך להיות ב-Authorization header

**אם רוצה להיות מוודא:**
```bash
# Check image URLs:
curl -s "https://omer4561.github.io/accident-native/admin.html" | \
  grep -i "img src" | head -5
# צריך להראות: src="https://huggingface.co/..."
# **לא:** src="https://...?token=..."
```

## שלב 7: בדוק OTP Rate Limiting

**בדוק בקונסול:**

```js
// בדוק שהקוד שדורות לא מתוקן:
const code = "123456";
const buf = await crypto.subtle.digest('SHA-256', new TextEncoder().encode('otp:' + code));
const hash = Array.from(new Uint8Array(buf)).map(b => b.toString(16).padStart(2,'0')).join('');
console.log(hash);

// עדיין לא צריך להתאים ל-codeHash שנשמר
```

## Troubleshooting

### "סיסמת מנהל לא הוגדרה"
```
❌ סיסמת מנהל לא הוגדרה — הוסף ADMIN_PASS ב-Secrets
```
**פתרון:** הוסף `ADMIN_PASS` secret ו-push מחדש.

### admin.html נטוען אבל לא מתחבר
1. בדוק שה-secret הוגדר
2. בדוק ש-build עבר בהצלחה (GitHub Actions)
3. בדוק שה-hash מוזרק: F12 → Console
   ```js
   console.log(ADMIN_PASS_HASH)  // לא צריך להיות '__ADMIN_PASS_HASH__'
   ```

### build-apk.yml failed: "KS_PASS is empty"
```
❌ סיסמת keystore לא הוגדרה — הוסף KEYSTORE_PASSWORD ב-Secrets
```
**פתרון:** הוסף `KEYSTORE_PASSWORD` secret ו-push מחדש.

### OTP לא מגיע
1. אם מייל: בדוק שـ `EMAILJS_PUBLIC_KEY` הוגדר
2. אם טלפון: בדוק שـ `TELEGRAM_TOKEN` תקין
3. בדוק spam folder

### XSS alert בפאנל הניהול
1. סביר שדוח עם payload בשדה metadata
2. בדוק את ה-developer console לשגיאות
3. בדוק שה-escaping קרה: F12 → בדוק את ה-HTML של הכרטיס

## עדכונים עתידיים

1. **Move auth to server** (cloud function / Vercel edge)
   - אבל לא הכרחי — כרגע בטוח מספיק עם hash-only

2. **נוסף database רמת סיסמה** (אם כמה מנהלים)
   - כרגע: סיסמה אחת בלבד

3. **Implement API key rotation** (HF_TOKEN, וכו')
   - בדוק מדי שלוש חודשים
