# הרשמת משתמשים ואימות

חסם חובה לאפליקציה: משתמש חדש חייב להרשם כדי להשתמש בכל תכונה. הרשמה כוללת אימות זהות דרך OTP.

---

## Modal הרשמה

### הפעלה
- מופיע בעלייה ראשונה של האפליקציה
- **חוסם את כל האפליקציה** עד הרשמה בוצעה בהצלחה
- כפתור "X" להסגרת modal לא זמין

### שדות קלט
1. **שם מלא** (חובה)
   - שם פרטי + משפחה
   - ערוך בפירוט עברי

2. **בחירת ערוץ אימות** (tabs)
   - 📧 **מייל** — Email
   - 📱 **טלפון** — Telegram (הודעה למפקד)

---

## OTP — אימות דרך מייל (EmailJS)

### Flow
1. משתמש מכניס שם + כתובת מייל
2. כפתור "שלח קוד"
3. **EmailJS** שולח מייל עם קוד 6 ספרות
4. משתמש מעתיק קוד מהמייל
5. כפתור "אמת"
6. בדיקה: hash של קוד חייב להתאים `codeHash` ב-localStorage

### Secrets Required
- `EMAILJS_SERVICE_ID` (שם השירות ב-EmailJS)
- `EMAILJS_TEMPLATE_ID` (template ID)
- `EMAILJS_PUBLIC_KEY` (GitHub Secret)

### הודעה
```
נשלח קוד לכתובת המייל שלך. הקוד בתוקף ל-10 דקות.
```

---

## OTP — אימות דרך טלפון (Telegram)

### Flow
1. משתמש מכניס שם + מספר טלפון
2. כפתור "שלח קוד"
3. **Telegram** שולח הודעה לערוץ הניהול (חייל עם CHANNEL_ID)
4. המפקד קורא את ההודעה: `פלוני אלמוני בטלפון 052-XXX-XXXX בקש אימות. קוד: 123456`
5. המפקד מעביר את הקוד לחייל (פרטי או הודעה קבוצתית)
6. חייל מכניס את הקוד באפליקציה
7. בדיקה: hash חייב להתאים `codeHash`

### Secrets Required
- `TELEGRAM_TOKEN` (token הבוט מ-@BotFather)
- `CHANNEL_ID` (ID ערוץ ניהול, לדוגמה `-1001234567890`)

### הודעה ללחץ
```
קוד נשלח למפקד. המתינו לקבלת הקוד.
```

## OTP — אחסון ו-Rate Limiting

### localStorage Format
```json
{
  "codeHash": "a3f5c2e...xyz",    // SHA-256('otp:' + code)
  "contact": "user@example.com",   // או "+972521234567"
  "email": "user@example.com",     // (מייל בלבד)
  "name": "יוסף כהן",
  "tries": 0,                      // counter
  "expiry": 1713132600000          // timestamp + 10 דקות
}
```

### אבטחה
- **אין plaintext code** לעולם — רק hash
- **שדה `tries`**: מעודכן בכל ניסיון אימות שגוי
- **Rate limiting**: מקסימום **5 ניסיונות** לפני מחיקה של `pending_otp`
- **TTL**: 10 דקות (600,000 ms)

### Hashing
```javascript
async function _hashOTP(code) {
  const buf = await crypto.subtle.digest('SHA-256', 
    new TextEncoder().encode('otp:' + code));
  return Array.from(new Uint8Array(buf))
    .map(b => b.toString(16).padStart(2,'0'))
    .join('');
}
```

### בדיקה בקונסול (DevTools)
```javascript
JSON.parse(localStorage.getItem('pending_otp'));
// Output:
{
  codeHash: "a3f5c2e8...",
  contact: "user@example.com",
  email: "user@example.com",
  name: "...",
  tries: 0,
  expiry: 1713132600000
}
```

## Profile Management

### אחסון לאחר הרשמה בוצעה
```json
localStorage.user = {
  "name": "יוסף כהן",
  "contact": "user@example.com",  // או מספר טלפון
  "email": "user@example.com",    // אם רישום בדוא"ל
  "registered_at": 1713132000000
}
```

### תצוגה בהגדרות
- פאנל "הגדרות" → "פרטים אישיים"
- שדות: שם, אימייל/טלפון
- כפתור: **"ערוך פרטים"** → modal רישום שוב (optional)

## Verification Flow (מלא)

### 1. משתמש חדש → Modal ראשי
```
[ שם מלא: __________ ]
[ 📧 מייל ] [ 📱 טלפון ]
```

### 2. בחרו 📧 מייל
```
[ שם מלא: יוסף כהן ]
[ כתובת אימייל: user@example.com ]
[ שלח קוד ]
```

→ EmailJS שולח מייל

### 3. קוד הגיע
```
[ הקוד שנשלח ל-user@example.com: ]
[ 123456 ]
[ אמת ]
```

### 4. אימות בוצע ✅
```
✅ ברוכים הבאים, יוסף!
```

localStorage.user נוצר, modal נסגר, אפליקציה נטען.

## וידוא טכני

### בעת sendOTP()
```javascript
if (email) {
  // EmailJS
  await emailjs.send(
    EMAILJS_SERVICE,
    EMAILJS_TEMPLATE,
    { to_email: email, code: code }
  );
} else {
  // Telegram
  const msg = `משתמש ${name} בטלפון ${phone} בקש אימות.\nקוד: ${code}`;
  await fetch('https://api.telegram.org/bot' + TG_TOKEN + '/sendMessage', {
    method: 'POST',
    body: JSON.stringify({
      chat_id: CHANNEL_ID,
      text: msg
    })
  });
}

// שמור hash בלבד
const codeHash = await _hashOTP(code);
localStorage.setItem('pending_otp', JSON.stringify({
  codeHash, contact, email, name, 
  expiry: Date.now() + 600000,
  tries: 0
}));
```

### בעת verifyOTP()
```javascript
const pending = JSON.parse(localStorage.getItem('pending_otp'));
const tries = (pending.tries || 0) + 1;

if (tries > 5) {
  localStorage.removeItem('pending_otp');
  _regErr(2, '⛔ יותר מדי ניסיונות — שלח קוד מחדש');
  return;
}

const enteredHash = await _hashOTP(userEnteredCode);
if (enteredHash !== pending.codeHash) {
  pending.tries = tries;
  localStorage.setItem('pending_otp', JSON.stringify(pending));
  _regErr(2, `קוד שגוי — נסה שוב (${5 - tries} נסיונות נותרו)`);
  return;
}

// הצלחה!
localStorage.removeItem('pending_otp');
localStorage.setItem('user', JSON.stringify({
  name: pending.name,
  contact: pending.contact,
  email: pending.email,
  registered_at: Date.now()
}));
```
