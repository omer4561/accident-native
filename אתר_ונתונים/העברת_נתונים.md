## Upload API — hfCommit()

### זרימה
1. משתמש שומר דוח בעלייה (מקומי MediaStore)
2. `_uploadGroupToHF()` קרא ברקע **לא חוסם UI**
3. `hfCommit()` מעלה ל-HuggingFace

### Function Signature
```javascript
async function hfCommit(files, folder) {
  // files = [
  //   { name: "report_12345_p1.jpg", blob: Blob },
  //   { name: "report_12345.json", blob: Blob },
  //   { name: "user_abc123.json", blob: Blob }
  // ]
  // folder = "data/user_abc123/"
  
  for (let file of files) {
    const path = folder + file.name;
    // HF Hub API: Upload file
    await fetch(
      `https://huggingface.co/api/datasets/${DATASET}/commit`,
      {
        method: 'POST',
        headers: { 'Authorization': `Bearer ${HF_TOKEN}` },
        body: JSON.stringify({
          path: path,
          blob: file.blob,
          ...
        })
      }
    );
  }
}
```

### Authorization
```javascript
// ✅ נכון:
headers: { 'Authorization': `Bearer ${HF_TOKEN}` }

// ❌ לא נכון:
url += `?token=${HF_TOKEN}`
```

## Background Upload — _uploadGroupToHF()

### Trigger
אחרי שמירה מקומית בוצעה:
```javascript
function _saveReport() {
  // 1. שמור ל-MediaStore.Downloads (מקומי)

  // 2. תיאום מקומי בלבד
  localStorage.setItem('reports', JSON.stringify([...]));
  
  // 3. (ברקע, לא חוסם)
  _uploadGroupToHF();
}
```

### Features
- **לא חוסם UI** — משתמש יכול להמשיך לסריקות נוספות
- **Retry logic** — אם network קדימה, retry בעוד שעה
- **Progress tracking** — optional localStorage flag

## Compression לפני Upload — compressForUpload()

### Parameters
```javascript
function compressForUpload(canvas) {
  // canvas.width ≤ 1200px (auto-scaled)
  // JPEG quality: 75% (0.75)
  
  canvas.toBlob(
    blob => { /* upload blob */ },
    'image/jpeg',
    0.75
  );
}
```

### יתרונות
- **~100-200KB per page** (מ-~1-2MB original)
- **קרא מספיק** להקראת מטא-נתונים
- **מהיר להעלאה** (network efficient)

## Read API — hfReadJson()

### Function Signature
```javascript
async function hfReadJson(path) {
  // path = "data/user_abc123/report_12345.json"
  
  const response = await fetch(
    `https://huggingface.co/datasets/omer4561/accident-reports/resolve/main/${path}`,
    {
      headers: { 'Authorization': `Bearer ${HF_TOKEN}` }
    }
  );
  
  if (!response.ok) throw new Error(response.status);
  return await response.json();
}
```

### בדיקה (Admin Panel)
```javascript
// בקונסול F12
const meta = await hfReadJson("data/user_abc123/report_12345.json");
console.log(meta);  // metadata object
```

## Android Bridge ממשק

### AndroidBridge Functions

#### saveFile()
```javascript
// Signature:
AndroidBridge.saveFile(base64Data, filename, subFolder)
// Stores: /Download/Accident_Reports/{subFolder}/{filename}

// Returns:
// "OK:Download/Accident_Reports/subfolder/filename.jpg"
// "ERROR:message"

// Example:
const result = AndroidBridge.saveFile(jpgBase64, "report_12345_p1.jpg", "");
if (result.startsWith("OK")) {
  console.log("Saved: " + result.substring(3));
} else {
  console.error("Save failed: " + result);
}
```

#### readFile()
```javascript
// Signature:
AndroidBridge.readFile(filePath)
// Reads: /Download/{filePath}

// Returns:
// Base64 string of file content
// "ERROR:message"

// Example:
const base64 = AndroidBridge.readFile("Download/Accident_Reports/report.jpg");
if (base64.startsWith("ERROR")) {
  console.error(base64);
} else {
  document.getElementById('image').src = 'data:image/jpeg;base64,' + base64;
}
```

#### scanFile()
```javascript
// Signature:
AndroidBridge.scanFile(path)
// Triggers: MediaScanner to index file for gallery

// Example:
AndroidBridge.scanFile("/Download/Accident_Reports/report.jpg");
```

#### getInstalledBuild()
```javascript
// Signature:
AndroidBridge.getInstalledBuild()
// Returns: build number of installed APK

// Example:
const build = AndroidBridge.getInstalledBuild();
if (data.apk > build) {
  // Update available
}
```

#### downloadAndInstall()
```javascript
// Signature:
AndroidBridge.downloadAndInstall(apkUrl)
// Downloads APK from URL + opens system installer

// Callbacks:
window._apkProg = function(percent) { /* 0-100 */ }
window._apkErr = function(error) { /* error message */ }

// Example:
AndroidBridge.downloadAndInstall(
  "https://github.com/.../releases/download/v42/app-release.apk"
);
```

## Data Flow דוגמה

### סימולציה: שמירה + Upload

#### 1️⃣ משתמש מסיים סריקה
```javascript
// 2 תמונות + metadata
const reportId = "report_12345";
const userId = "user_abc123";

// Compress + encode to base64
const p1_base64 = await compressAndEncode(page1Canvas);
const p2_base64 = await compressAndEncode(page2Canvas);

const metadata = {
  id: reportId,
  userId: userId,
  plate: "123-45-678",
  driver: "David Cohen",
  urls: [
    `https://huggingface.co/datasets/omer4561/accident-reports/resolve/main/data/${userId}/${reportId}_p1.jpg`,
    `https://huggingface.co/datasets/omer4561/accident-reports/resolve/main/data/${userId}/${reportId}_p2.jpg`
  ],
  timestamp: Date.now()
};
```

#### 2️⃣ שמור מקומי (מיד)
```javascript
// MediaStore
const r1 = AndroidBridge.saveFile(p1_base64, reportId + "_p1.jpg", "");
const r2 = AndroidBridge.saveFile(p2_base64, reportId + "_p2.jpg", "");

// localStorage (backup)
localStorage.setItem(reportId, JSON.stringify(metadata));

// UI
showMessage("✅ הדוח נשמר בהצלחה");
```

#### 3️⃣ Upload ברקע (async, לא חוסם)
```javascript
// _uploadGroupToHF() קרא בחסמיסה
async function _uploadGroupToHF() {
  const files = [
    { name: reportId + "_p1.jpg", blob: p1_blob },
    { name: reportId + "_p2.jpg", blob: p2_blob },
    { name: reportId + ".json", blob: new Blob([JSON.stringify(metadata)]) },
    { name: `../users/${userId}.json`, blob: new Blob([JSON.stringify(userMetadata)]) }
  ];
  
  try {
    await hfCommit(files, `data/${userId}/`);
    console.log("✅ Upload to HF complete");
  } catch (err) {
    console.error("Upload failed, will retry later:", err);
    // Retry logic ב-next session
  }
}
```

#### 4️⃣ Admin קורא הדוח (later)

## Error Handling

### Upload Retry
```javascript
async function hfCommitWithRetry(files, folder, maxRetries = 3) {
  for (let i = 0; i < maxRetries; i++) {
    try {
      return await hfCommit(files, folder);
    } catch (err) {
      if (i < maxRetries - 1) {
        console.log(`Retry ${i + 1}/${maxRetries} in 5s...`);
        await new Promise(r => setTimeout(r, 5000));
      } else {
        throw err;
      }
    }
  }
}
```

### Network Detection
```javascript
if (!navigator.onLine) {
  console.log("No network, will retry when online");
  window.addEventListener('online', _uploadGroupToHF);
  return;
}
```

## Security

### Authorization Header
- **תמיד Bearer token**, לא query parameter
- **Whitelist URLs** ב-admin panel (`https://huggingface.co/`)
- **Escape metadata** לפני insertion ל-HTML

### File Size Limits
- Per file: ~50MB (HF limit)
- Compressed JPGs: 100-200KB (safety margin)
